# 🚀 因果特征路由机制 (Causal Feature Routing) 全景架构文档

## 🌟 核心背景 (Background)

在推荐系统与广告的 Uplift Modeling 中，引入因果中介变量（Mediator，如点击率、领券概率）可以极大提升对头部高潜客群的召回。然而，传统的融合范式（如 CTCVR 的串联或 MTL 的并联）通常会引发严重的**负迁移 (Negative Transfer)**——中介模型对“羊毛党 (Always-Taker, AT)”的高估偏置，会污染下游的 Uplift 预测，导致大盘 `5%-20%` 区间转化率崩盘，尾部排序混乱。

为了解决该痛点，我们提出了**因果特征路由网络 (Causal Feature Routing)**。其核心理念是：**打破对中介信息的盲目服从，将其降级为“可被质疑的软先验 (Soft Prior)”，并允许底层特征网络 $h(X)$ 动态地修正、拒绝或增强这一先验。**

本架构通过定义**两大先验空间底座 (V8/V11)**、**三种空间对齐算法**以及**八大信息交互方案 (S1-S8)**，构建了业界最严谨的因果纠偏消融矩阵。

## 🏛️ 第一维度：先验空间底座 (The Base Space)

这一维度回答的核心科学问题是：*特征网络究竟应该在什么数学空间下与中介先验进行博弈？*

### 1. V8 架构：原始概率空间流 (Raw Probability Space)
* **机制简介**：直接接收被冻结的中介模型 (Model C) 输出的原始概率 $[P_{AT}, P_{CO}, P_{NT}]$。
* **空间特征**：值域被死死限制在 $[0, 1]$ 之间，且大盘分布极度不平衡（例如 $P_{CO}$ 的期望可能仅为 $0.002$）。
* **作用方式**：网络通常只能在高度折叠的概率空间中进行**“乘法打折”**（衰减）。防守极强（能把羊毛党的概率乘 0 截断），但存在严苛的“先验天花板”，攻击打捞能力受限。

### 2. V11 架构：EMA 动态 Logit 空间流 (Aligned Logit Space) 🌟 *终极推荐*
* **机制简介**：摒弃原始概率，引入 `DynamicPriorAlignmentLayer`。在训练期使用 **EMA (指数滑动平均)** 动态追踪大盘真实的先验期望均值与方差。
* **作用方式**：抹平大盘的不平衡偏置。网络在同一起跑线上输出无边界的偏移量 $\Delta L$ 进行**“加法融合”**，彻底打破先验天花板，既能“踩刹车”拦截羊毛党，也能“踩油门”提拔被低估的隐藏金子。

#### 📐 V11 的三大核心空间对齐算法 (Alignment Types)：
1. **Lift (对数几率 / Log-Odds) 🌟**：
   * **公式**：$L_{prior} = \ln \left( \frac{P_{sample}}{P_{global\_mean}} \right)$
   * **物理意义**：“该样本的潜力是大盘均值的几倍？” 将不对称的概率拉伸为以 0 为中心的 $(-\infty, +\infty)$ 连续分布。对长尾极低概率的微小波动极其敏感，是工业界最鲁棒的选择。
2. **Z-Score (动态高斯标准化)**：
   * **公式**：$L_{prior} = \frac{P_{sample} - P_{global\_mean}}{Std_{global}}$
   * **物理意义**：“该样本偏离了大盘几个标准差？” 暴力放大分布的极端性，用最剧烈的手段撕开金子与羊毛党的界限，适合中介模型打分极其极化的情况。
3. **Rank (Logit 逆映射排位)**：
   * **公式**：$L_{prior} = \ln \left( \frac{rank}{1 - rank} \right)$ （其中 rank 为 Batch 内的百分位 $0 \sim 1$）
   * **物理意义**：“内卷排位赛”。无论大盘数值发生多剧烈的绝对漂移，只关心该样本在当前批次中的相对次序。排名前 1% 的人拿到 $+4.59$ 底分，垫底的拿到 $-4.59$ 底分。彻底免疫绝对数值漂移的终极防线。

## 🧬 第二维度：八大信息交互方案 (The Fusion Schemes)

这一维度展示了特征网络 $h(X)$ 修正中介先验的演进史，回答了：*网络应该以何种姿态进行决策？*

### 🟡 早期探索组：软截断与门控 (S1, S2, S3)
*这类方案试图在概率空间内通过设定阈值来进行截断，是演进过程中的中期产物。*
* **S1 (动态阈值软截断)**：预测一个 1D 偏移量 $\Delta C$。基于预设的 Top 5% 阈值，将计算出的 $\Delta C$ 作用于阈值上，实现千人千面的动态门槛。
* **S2 (单门控特征驱动)**：预测一个全局附加分 $V$，与缩放后的先验残差相加后经过 Sigmoid。所有专家共用一个门控开关。
* **S3 (独立多门控驱动)**：S2 的进阶版，预测 3 个独立的附加分 $V$，赋予 AT、CO、NT 三个专家塔各自独立的激活阈值。

### 🟢 纯特征独立组 (S4, S5)
*抛弃硬性阈值，赋予底层特征绝对的自由度。*
* **S4 (独立特征乘法门控)**：【**闭门造车的审核员**】。特征网络在完全看不到中介先验的情况下，独立给出置信度 $\sigma(MLP(X))$，与先验概率直接相乘。赋予了网络一票否决权，防守强但无法突破先验上限。
* **S5 (纯 Softmax MoE)**：抛弃一切先验锚点，纯靠 $X$ 输出 Softmax 权重。为了防止初期乱跑，引入了第 4 个“虚拟兜底专家 (Dummy Expert)”吸走绝大部分初始概率，后续随训练逐渐将权重交还给真实专家。

### 🔴 终极战力组：高级因果路由 (S6, S7, S8) 🌟 *论文核心打擂组*
*最符合深度学习数学直觉与因果逻辑的高级融合方案。*

* **S6 (Logit 空间加法融合)**：【**无边界的修正补丁**】
  * **机制**：特征网络输出无边界的连续值 $\Delta L$，在 Logit 空间中与先验 Logit $L_{prior}$ 直接相加，最后过 Softmax 分配概率。
  * **表达**：$P_{final} = Softmax( L_{prior} + MLP(X) )$
  * **评价**：S4 的终极进化版。正数踩油门，负数踩刹车，完美解决概率空间天花板问题。

* **S7 (动态信任插值)**：【**看情况甩锅的聪明滑头**】
  * **机制**：特征网络不仅输出自己的独立预测 $P_{feat}$，还输出一个全局置信度标量 $\alpha \in [0, 1]$。最终结果为纯先验与纯特征预测的**凸组合 (Convex Combination)**。
  * **表达**：$P_{final} = \alpha \cdot P_{prior} + (1 - \alpha) \cdot P_{feat}$
  * **评价**：赋予模型“自适应退化能力”。底层特征模糊时听从先验（吃红利）；发现先验离谱时强行夺权。对特征分布的抖动极其鲁棒。

* **S8 (先验感知注入 + 乘法)**：【**偷看长官底牌的参谋**】
  * **机制**：在做决策前，将中介先验概率（或对齐后的 Logit）直接拼接（Concat）到底层特征 $X$ 之后。网络结合自身特征与先验打分，综合输出最终修正门控。
  * **表达**：$P_{final} = P_{prior} \times \sigma( MLP( X \oplus Prior ) )$
  * **评价**：具备强上下文感知 (Context-aware) 能力。网络能够根据“先验给的分值高低”来动态调整自己的打折力度，显著增强了中介与特征网络的化学反应。

## 📊 论文消融实验 Storyline 建议 (How to write the paper)

当你拿着上述模型的测试结果（AUUC, Qini 等）时，可以通过以下逻辑线构建无懈可击的消融实验分析：

1. **证明空间重构的必要性 (V8 vs V11)**：
   * 对比 `V8_S4` 与 `V11_S4_Lift`。论证“将高度不平衡的概率拉伸至 Logit 空间并抹平大盘偏置 (EMA Alignment)”，是引入因果中介、避免网络初期梯度“饿死”的先决条件。
2. **防守与攻击的权衡 (S4 vs S6)**：
   * 对比 `V11_S4` (乘法衰减) 与 `V11_S6` (Logit加法)。通过 S6 在尾部的压制与头部的爆发，论证纯防守型的乘法门控会造成“天花板效应”，而无边界的加法流能够有效“打捞”高质量长尾客群。
3. **空间对齐算法的敏感性 (Lift vs Z-Score vs Rank)**：
   * 横向对比 `V11_S6` 在不同对齐方式下的表现。解释在当前数据集的分布特性下，为何基于相对比例的 Lift (或基于绝对排位的 Rank) 会比传统的 Z-Score 更具工业鲁棒性。
4. **信息交互深度的价值 (S4 vs S7 vs S8)**：
   * 对比独立运算的 S4 与具备“信任分配”的 S7、“上下文感知”的 S8。论证黑盒式的盲目打折不如赋予网络“甩锅权”或“知情权”来得高效，以此证明因果纠偏路由机制设计的合理性。

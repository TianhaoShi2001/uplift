# 🚀 Uplift 靶向纠偏架构演进史 (The Uplift Ablation Matrix)

> **核心背景**：本系列模型基于 TARNET 架构，旨在解决推荐系统 Uplift 建模中由于流行度偏差（Popularity Bias）导致的“高先验假阳性”（羊毛党）误判问题。核心目标是填平 `5%-10%` 头部区间的 ATE（平均因果效应）亏损。

---

## 🧱 Phase 1: 基础融合与基线探索 (V1 - V5)
*早期验证，确立了将倾向分（Propensity Score）或先验作为干预特征的有效性与缺陷。*

* **`y_v1_base` (纯净基线)**：无 Model C 融合，纯特征拟合 + BCE Loss。（羊毛党重灾区，Baseline）
* **`y_v2_emb` (特征拼接)**：将 Model C 输出的 $\pi$ 转化为 Embedding 拼接入底座。
* **`y_v3_moe` (因果多头路由)**：构建 3 个独立专家塔，用 $\pi$ 作为门控软路由。（存在参数爆炸与小群体过拟合问题）
* **`y_v4_loss_strata` (靶向重加权)**：网络退回 V1，但在 Loss 侧用 $\pi_{01}$ 作为样本权重。
* **`y_v5_loss_var` (方差正则化)**：引入额外 Loss，强迫非敏感客群（Always-taker & Never-taker）的 Uplift 趋近于 0。

---

## 🏗️ Phase 2: 架构定型与物理隔离 (V6 - V8)
*解决 V3 的结构缺陷，通过前向传播（Forward）的机制设计，将噪音在网络结构层面隔离。*

### 🌟 V6: 残差 MoE (Residual MoE)
* **核心动作**：所有后续高级版本的**通用标准底座**。主塔学习大盘共性知识，极轻量的“残差专家 MLP”仅负责学习特定人群的增量偏差。彻底解决了参数爆炸。

效果： 虽然总体AUUC相比V3差，但是头部lift，AUUC都更好，是随机的尾部拖底了，和V3一样，尾部是0.5左右。另外，对中介的依赖程度更低。

### 🛡️ V7: 排序感知截断 (Rank-Aware Truncated MoE)
* **核心动作**：结构侧**极强防守**。基于巨型 Batch 内的大数定律，强行切断非 Top K% 流量的专家特权。
* **子版本**：
  * `hard_top5` / `hard_top10`：硬截断，过线为 1，不过线为 0。
  * `soft_top5` / `soft_top10`：软截断，带有温度系数的 Sigmoid 平滑过渡（防止梯度突变）。

### 🧬 V8: 动态演进门控 (Evolution MoE)
* **核心动作**：不再盲信 Model C 的死板先验，而是让网络利用底层特征 $h(X)$ 动态生成或修正门控（Gate）。
* **子版本详解**：
  * **`s1` (动态阈值降分)**：预测一个 1D 偏移量 $\Delta C$，对原始先验的截断阈值进行动态平移。
  * **`s2` (单门控特征驱动)**：预测一个全局附加分 $V$，与先验残差相加后过 Sigmoid。
  * **`s3` (独立多门控) -> 🌟 推荐**：预测 3 个独立的附加分，分别调控 3 个专家的先验偏差。
  * **`s4` (特征 MLP门控)**：靠底层特征输出 3 维 Logit 作为门控。
  * **`s5` (特征 MLP Softmax 权重)**：抛弃先验锚点，纯学习权重。

---

## ⚔️ Phase 3: 终极纠偏惩罚 (V9 - V10)
*在 V6 的强力底座上，专治 5-10% 羊毛党的反向传播（Backward）魔法。通过 Loss 设计逼迫模型认错。*

### 🎯 V9: 靶向 Group DRO (稳健隐式优化)
* **核心动作**：上帝闭眼。不知道谁是坏人，只知道哪个群体的整体 Loss 崩了，就用指数级 EMA 动态放大该组权重（走火入魔式逼迫）。
* **子版本详解**：
  * **`a_coarse` (1D 粗切 4 组)**：仅按先验 $\pi_{01}$ 切分为 0-5%, 5-10%, 10-20%, 20-100%。
  * **`b_coarse` (2D 粗切 16 组) -> 🌟 杀招**：4 组先验 $\times$ 4 种因果事实 $(T, Y)$。精准隔离出 `[5-10%] ∩ [T=1, Y=0]` 的羊毛党毒药桶。
  * **`a_fine` (1D 细切 11 组)**：按十分位（外加前 5% 单切）细分，验证精度下探。
  * **`b_fine` (2D 细切 44 组)**：11 组先验 $\times$ 4 种因果事实。用作反证对照组，证明过度细分会导致方差爆炸。

### 👁️‍🗨️ V10: 先验冲突代价敏感 (上帝视角显式纠偏)
* **核心动作**：上帝睁眼。精确计算干预组 ($T=1$) 中，真实标签 $Y$ 与先验 $\pi_{01}$ 的“物理冲突度”，直接对单体样本施加极刑惩罚或重赏。
* **子版本详解**：
  * **`wool_only` (仅惩罚羊毛党)**：对于 $T=1, Y=0$ 的人，先验越高，惩罚越重。专项填平 5-10% 亏损。
  * **`gold_only` (仅奖励隐藏金子)**：对于 $T=1, Y=1$ 的人，先验越低，权重加得越大。专项打捞被先验看扁的优质用户。
  * **`both` (双管齐下)**：同时开启 `wool_only` 和 `gold_only`，双向全面纠偏。

---

## 🔗 Phase 4: 终极缝合怪 (The Ultimate Combos)
*最能打、论文实验最有说服力的版本：结构侧防守 + 优化侧进攻。先用软门控筛掉 95% 大盘噪音，再在核心战区用 Loss 实施精准打击。*

### 🛡️ + ⚔️ [V7 + V9]: 软截断门控 + 靶向 Group DRO
*使用 V7 的 `soft_top5` (截断非前 5% 流量) 作为底座，配合 V9 的动态重加权。*
* **`y_v7_dro_a_coarse`**：V7 Soft Top 5% + DRO 1D 粗切 4 组
* **`y_v7_dro_b_coarse`**：V7 Soft Top 5% + DRO 2D 交叉 16 组 (🌟 结构与损失的巅峰配合)
* **`y_v7_dro_a_fine`**：V7 Soft Top 5% + DRO 1D 细切 11 组
* **`y_v7_dro_b_fine`**：V7 Soft Top 5% + DRO 2D 细切 44 组

### 🛡️ + 👁️‍🗨️ [V7 + V10]: 软截断门控 + 先验冲突显式加权
*使用 V7 的 `soft_top5` 作为底座，配合 V10 的单体显式代价敏感惩罚。*
* **`y_v7_conflict_wool`**：V7 Soft Top 5% + 仅惩罚 5-10% 漏网羊毛党
* **`y_v7_conflict_gold`**：V7 Soft Top 5% + 仅打捞漏网金子
* **`y_v7_conflict_both`**：V7 Soft Top 5% + 靶向双重纠偏 (🌟 理论最强、且最具可解释性的终极杀器)
